= Home Page Redirector plugin for Redmine

A very simple redmine plugin that will redirect signed-in users to My Page when they sign-in or when they click the home link in the website header.

Tested with Redmine 5.0.5.stable

Install to redmine/plugins directory. No migrations are needed. No configuration needed 


Copy /lib/home page redirect.rb to /redmine/app/models
